# Installation

## Stable release

To install Update Burden, run this command in your terminal:

```sh
uv add update-burden
```

Or if you prefer to use `pip`:

```sh
pip install update-burden
```

## From source

The source files for Update Burden can be downloaded from the [Github repo](https://github.com/mklymyshyn/ossiq-cli).

You can either clone the public repository:

```sh
git clone git://github.com/mklymyshyn/ossiq
```

Or download the [tarball](https://github.com/mklymyshyn/ossiq-cli/tarball/master):

```sh
curl -OJL https://github.com/mklymyshyn/ossiq-cli/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd ossiq
uv pip install .
```
