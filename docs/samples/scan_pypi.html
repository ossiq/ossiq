Legend

Drift Status: MAJOR

Your dependency is one or more **major versions** behind, implying accumulated breaking changes and higher remediation effort if addressed.

Drift Status: MINOR

Behind on non-breaking feature releases; remediation is typically low-risk and can be planned opportunistically.

Drift Status: PATCH

Missing bugfixes or security patches; remediation effort is usually minimal.

Drift Status: LATEST

Aligned with the latest release; no remediation required at this time.

stacks
Release Distance

The number and type of releases between the installed version and the latest available version. e.g., 1.0.0 to 1.0.1 is 1 patch release, 1.0.0 to 1.1.0 is 1 minor release, 1.0.0 to 2.0.0 is 1 major release.

change_history
Time Lag

The time elapsed between the installed version and the latest available release. Color indicates how far the dependency has drifted relative to configured thresholds. Green: less than 365 days, Orange: more than 365 days, Red: more than 730 days.
